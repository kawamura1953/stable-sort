ssortのベンチマークテストの要領

１．github.com/kawamura1953/stable-sort から次の４つのファイルをダウンロードする。

    ss14g1.c     
    qs_glibc.c

    benchmark.sh 


２．benchmark.sh を実行する。実行結果は benchmark.txt へ書き込まれる。

　　main_prog.c（benchmark.shの中でコンパイル・実行）の引数と出力の説明は、
　　Readme2.txtを参照。


３．その他のファイル

    Readme.txt    : このファイル
